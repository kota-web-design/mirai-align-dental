# MIRAI ALIGN DENTAL LP

## 概要

マウスピース矯正専門の架空クリニック「MIRAI ALIGN DENTAL」の1ページLPです。無料相談、Web予約、LINE相談への導線強化を目的に制作しています。

## ターゲット

- 20〜40代の男女
- 目立たない矯正を検討している人
- 費用や期間が不安な人
- 安さだけでなく、診断やサポートも重視したい人

## 制作意図

参考カンプの「清潔感」「透明感のあるブルー」「カード型UI」「スマホでCVしやすい縦長構成」を参考にしつつ、テキスト、画像、ロゴ、細かいレイアウトはオリジナルで再設計しました。

FVでは「目立たない、取り外せる」というわかりやすい訴求を置き、直後に安心材料とCTAを配置。中盤では安さだけで選ぶリスク、診断・症例・料金・流れを順番に見せ、最後に電話/LINE/Web予約の3導線へつなげています。

## デザインのポイント

- メインカラーは透明感のあるブルー系
- 余白は広めに取りつつ、SPではCVまで読み進めやすい密度に調整
- 料金部分は控えめなゴールドで視線誘導
- カードUIは角丸8px、やわらかいシャドウで統一
- SPでは下部固定CTAを表示
- PCでは横幅を活かし、FVとCTAが広い画面でも窮屈に見えないよう調整
- FVと最終CTAはPC/SPで背景画像を分け、コードのテキスト・ボタンを重ねる構成にしています
- FVと最終CTAの小さな機能アイコンは、用途が伝わりやすいシンプルなSVG線画で調整
- 悩み・注意喚起・治療フローには、やわらかい線画PNGアイコンを使用

## 架空サイトであること

本LPはポートフォリオ掲載用の架空サイトです。クリニック名、住所、料金、症例、医師プロフィールはすべてサンプルです。

## 医療広告表現への配慮

以下のような断定的・過度な表現は避けています。

- 必ず治る
- 絶対に失敗しない
- 最安
- No.1
- 効果保証

症例・料金には架空サンプルであること、効果には個人差があること、リスク・副作用の可能性があることを注記しています。
